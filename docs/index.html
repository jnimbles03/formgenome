AI-powered PDF form analysis. Right from your browser.

Download for Chrome
Version 2.0.0 GitHub

Form Experience Conversion Analysis.

Form Genome takes any URL — a marketing site, a single PDF form, a deep link to one field — and crawls politely outward to find every form on the domain. Each form is scored on field count, page count, conditional logic, calculations, and NIGO risk using a vision-LLM pipeline.

The output is a single self-contained index.html dashboard with cost estimation, complexity quadrants, and five analysis tabs. Open it, share it, or embed it.

How it works

  1. Crawl. robots.txt-respecting, per-host limited, honest UA.
  2. Analyze. Per-form vision-LLM scoring.
  3. Render. One HTML file. No backend dependency to view.

The Chrome extension v2.0.0

For day-to-day work, the extension is the better surface. It auto-detects PDF links on any page you visit, scores complexity, fields, entities, and action types in place, and queues batches across multi-page listings. Smart filtering hides guides and disclosures so you only see real forms.

Currently distributed via GitHub releases — download the zip, unzip, then load the chrome-extension folder via chrome://extensions with Developer mode on. Chrome Web Store listing in review.
