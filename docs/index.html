Form Genome

AI-Powered PDF Form Analyzer

Now available as a Chrome Extension

Download for Chrome

Form Genome has evolved. What used to be a web app is now a Chrome extension for a faster, more seamless experience. Install it once and analyze PDF forms from any webpage — no more switching tabs.

What it does

PDF Detection

Automatically finds PDF links on any page you visit. No manual searching required.

AI Analysis

Instant complexity scoring, field counts, entity detection, and action type classification.

Team Dashboard

Collaborative view of all team-analyzed forms with filtering and CSV export.

Batch Processing

Analyze multiple PDFs at once with real-time progress tracking.

Smart Filtering

Auto-detect and filter out guides, disclosures, and marketing docs.

Deep Scan

Scan neighboring pages to discover more PDFs across multi-page listings.

Install in 5 minutes

1

Download the extension

Grab the latest .zip from the GitHub Releases page.

2

Unzip the file

Extract the ZIP — you should see a folder called chrome-extension.

3

Open Chrome Extensions

Navigate to chrome://extensions and toggle on Developer mode in the top-right corner.

4

Load the extension

Click "Load unpacked" and select the extracted chrome-extension folder.

5

Pin & go

Click the puzzle icon in Chrome's toolbar and pin Form Genome. You're all set!
