Privacy Policy

Last updated: February 18, 2026

Form Genome ("the Extension") is a browser extension that analyzes PDF forms using AI. This policy explains what data we collect, how it's used, and your choices.

1. What We Collect

When you use the Extension to analyze a PDF, the following data is sent to our analysis server:

  • PDF URL — the web address of the PDF you choose to analyze
  • PDF file content — the document is sent to our server for AI processing
  • A locally-generated user ID — a random identifier (not linked to your name, email, or any personal information) used to associate analyses with your session

2. What We Do NOT Collect

  • We do not collect your name, email, or any personal contact information
  • We do not collect your browsing history or track which websites you visit
  • We do not read or access any page content unless you explicitly click "Analyze"
  • We do not sell, share, or transfer any data to third parties
  • We do not use cookies or tracking technologies

3. How Data Is Used

PDF content is sent to our secure server solely to perform form analysis (field detection, complexity scoring, entity classification). Analysis results are:

  • Stored locally in your browser's extension storage
  • Synced to a shared team dashboard (if team features are enabled) so team members can see aggregated form analysis results

4. Data Storage & Security

  • Analysis results are cached locally in your browser using chrome.storage.local
  • Server communication uses HTTPS encryption
  • Our analysis server is hosted on Google Cloud Run with standard security controls
  • No data is stored on the server longer than necessary to complete the analysis

5. Permissions Explained

  • activeTab — allows the Extension to read PDF links on the page you're currently viewing, only when you click the extension icon
  • scripting — allows the Extension to scan the active page for PDF links when you open the popup
  • storage — allows the Extension to save your analysis results and preferences locally in your browser

6. Your Choices

  • You can clear all cached analysis data from the Extension popup at any time
  • You can uninstall the Extension to remove all locally stored data
  • No analysis is performed unless you explicitly trigger it

7. Changes to This Policy

If we make material changes to this policy, we will update the "Last updated" date above. Continued use of the Extension after changes constitutes acceptance.

8. Contact

For questions about this privacy policy, please open an issue on our GitHub repository.
